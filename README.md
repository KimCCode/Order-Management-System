# Order Management System

This is an **Order Management System** developed in Java using Swing for the graphical user interface (GUI). The application is packaged as an executable JAR file.

## Features
- Manage customer orders.
- View, edit, and delete order details.
- Graphical user interface for ease of use.

## Requirements
To run this application, you need the following:
- Java Runtime Environment (JRE) 8 or higher installed.
  - You can download the latest JRE from [Oracle's official website](https://www.oracle.com/java/technologies/javase-jre8-downloads.html) or use any OpenJDK distribution.

## Download
1. Go to the [releases page](https://github.com/your-username/your-repository/releases).
2. Download the latest version of `app.jar`.

## Running the Application

Once you've downloaded the JAR file, follow these steps to run the application:

### macOS/Linux
1. Open a terminal.
2. Navigate to the folder where you downloaded `app.jar`.
   ```bash
   cd path\to\folder
3. Run the application:
```bash
   java -jar app.jar
